Visualizing Yelp Data

In Python, I created a scalable map visualization of predicted restaurant ratings using machine learning, the Yelp academic dataset for Berkeley restaurants, and D3.js visualization tool. To predict restaurant ratings, I developed an algorithim using object-oriented data abstraction, unsupervised learning (k-means algorithm), and supervised learning (SLS regression). In the visualization above, Berkeley is segmented into regions, where each region is shaded by the predicted rating of the closest restaurant (yellow is 5 stars, blue is 1 star). Specifically, the above visualization I constructed is a Voronoi diagram. I also used machine learning and list comprehensions to model, filter, and organize the Yelp academic database (5 datasets, ~10 GB total).
